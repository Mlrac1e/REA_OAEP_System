# RSA_OAEP_System
 基于pyQt6的RSA-OAEP加密系统
## 功能
 生成RSA公私钥对
    加密消息
    解密消息

## 文件结构
```
├── README.md
├── __pycache__
├── main.py
├── DecryptionWindow.py
├── EncryptionWindow.py
├── requirements.txt
```

## 环境配置
安装python环境
```
    conda create -n RSA_OAEP python=3.9 
```
激活python环境
```
    conda activate RSA_OAEP 
```
安装程序环境
```
    pip install -r requirements.txt 
```
